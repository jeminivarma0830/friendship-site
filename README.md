Description

Friendship Site is a web-based platform designed to help people connect, interact, and build meaningful friendships online. Users can create profiles, discover new people, send friend requests, and communicate through messaging features. The platform focuses on simplicity, security, and a smooth user experience.

🚀 Features
👤 User Registration & Login
📝 Profile Creation & Customization
🤝 Send / Accept Friend Requests
💬 Chat / Messaging System
🔍 Discover New Friends
📱 Responsive Design (Mobile Friendly)
